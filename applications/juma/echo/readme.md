echo是一个嵌入式固件。

其功能是通过蓝牙协议想硬件发什么，硬件将受到的原数据回传。

你可以通过其来检测软硬件蓝牙通讯是否正常可用。

双击mdk-arm\CAF.uvprojx即可运行程序。
